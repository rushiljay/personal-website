IBM Data Science Capstone Project

Source Code

Use various data science methods to find the best place to open a boba tea shop in the San Francisco Bar Area. Methods used in this project include k means classification and clustering, jupyter notebooks, and scikit learn. Used geospatial visualizations to show population density and already existing boba shops.

Grocery List AI App

Source Code

A sophicticated grocery list application that uses roberta-base transformer model trained with the Instacart Grocery Item Departments dataset to predict the category of each grocery item. This is hosted on a FastAPI backend with a React.JS front end, which includes features such as copy list, manipulate item quantity, and much more.

Simvestor: Stock Paper Trading Java App

Source Code

This is a paper trading app on trading stocks using fictional (paper money) built using Java Swing graphics. This was a capstone of the JAVA Honors course at Homestead High School, California. We used object-oriented programming skills and animation, and saving to csv files to keep track of trades and test new stratgies safely.

Attendr: In Flight Buddy leveraging Local LLMs

Source Code

This is a React-Native App that uses FastAPI as well as Mistral7B LLM running locally through Ollama for a in-flight helper application. Chatbot can answer basic questions about the flight, in-flight menu options, bathroom facilities, and ways to request help. This project was completed as part of TAMU Hack 2024 for the American Airlines Challenge.
